Android Studio 导入 jar 包及第三方开源库教程

可分为以下三种情况：

  1. jar
    将要添加的 jar 包放在项目的 module (默认叫app) 的 libs 下，右键 jar 包，选择 “Add As Library…”，添加成功。
  • 远程开源库
    所谓远程开源库，指你不需要下载下来，只要添加一句依赖，Android Studio 自动帮你下载。远程开源库的添加方式很简单，在 github 上找到此开源库，在开源库的下面说明文档中有添加 dependencies 的方 式。如果没有，就不能使用这种方式添加。把红色标记的依赖添加到 build.gradle（module: app）中（注意：不是 build.gradle(Project:..）），别忘了 “Sync Now” 同步一下。

  • 本地开源库
    将开源库下载下来，放在与 app 同级目录下，例如我放的一个开源库叫 “volley” ，然后在 setings.gradle 文件中，这样添加:

    include ':app',':volley'

    再回到 build.gradle（module : app） 中，在 dependencies 添加依赖:

    compile project(':volley')

    就添加成功了。
